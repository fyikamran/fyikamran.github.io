## Why Markdown is Clean

Markdown allows me to focus purely on the text rather than wrestling with code. The Jekyll engine compiles it into rich text instantly.

### Core Markdown Formats to Remember:

1. **Bold Text:** Wrap words in double asterisks like **this**.
2. *Italic Text:* Wrap words in a single asterisk like *this*.
3. **Links:** Put the text in brackets and the URL in parentheses: [Check out my homepage](https://fyikamran.com).
